Sélecteurs et Spécificité :

Les sélecteurs sont utilisés pour cibler les éléments HTML que vous souhaitez styliser. Les principaux types de sélecteurs comprennent : Sélecteurs de type : cibler les éléments par leur nom de balise (par exemple, p, h1). Sélecteurs de classe : cibler les éléments par leur attribut de classe (par exemple, .classe). Sélecteurs d'ID : cibler les éléments par leur attribut d'ID (par exemple, #id). Sélecteurs d'attribut : cibler les éléments en fonction de la valeur d'un attribut (par exemple, [type="text"]). Spécificité et Inheritance : Comprendre la spécificité est crucial pour déterminer quel style est appliqué lorsqu'il y a des conflits. La spécificité dépend des types de sélecteurs utilisés. Les styles peuvent également être hérités de parents à enfants, ce qui peut influencer l'apparence des éléments.

Propriétés et Valeurs de Base :

Propriétés de Couleur et de Texte : Couleur : color, background-color. Texte : font-family, font-size, font-weight, line-height, text-align, text-decoration. Modèle de Boîte CSS : Le modèle de boîte est fondamental pour comprendre comment les éléments sont espacés et dimensionnés. Padding : espace à l'intérieur de la boîte, entre le contenu et la bordure. Bordure : ligne entourant la boîte. Marge : espace à l'extérieur de la boîte. Dimensions : width, height.

Mise en Page :

Le positionnement contrôle la disposition des éléments sur la page. Position statique : position par défaut des éléments. Position relative : position en relation à sa position normale. Position absolue : position en relation à l'élément parent positionné le plus proche. Position fixe : position en relation à la fenêtre du navigateur. Position sticky : une combinaison de position relative et fixe. Flexbox et Grid : Ces deux modules CSS modernes facilitent la création de mises en page complexes. Flexbox : Utilisé pour disposer les éléments en ligne ou en colonne avec des propriétés comme display: flex, justify-content, align-items. Grid : Utilisé pour créer des mises en page bidimensionnelles avec des propriétés comme display: grid, grid-template-columns, grid-template-rows.

Réactivité et Media Queries :

Les media queries permettent de créer des mises en page réactives qui s'adaptent à différentes tailles d'écran. Syntaxe de base : @media (min-width: 768px) { ... } Breakpoints : points spécifiques où la mise en page change pour s'adapter à différentes tailles d'écran. Techniques Réactives : Utilisation de pourcentages ou unités relatives (em, rem, vw, vh) pour les tailles. Design mobile-first : créer des styles de base pour les petits écrans, puis ajouter des styles pour les écrans plus grands.

Animations et Transitions :

Les transitions permettent d'ajouter des effets de changement en douceur entre les états des éléments. Propriété de transition : transition: property duration timing-function delay; Animations CSS : Les animations permettent de créer des effets complexes en définissant des étapes clés. @keyframes : définit les étapes de l'animation. Propriété animation : animation: name duration timing-function delay iteration-count direction;
